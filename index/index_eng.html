Useful app for Minecraft

This page has homemade useful applications that can be used in Minecraft.

Minecraft Chunk Border Checker

This page calculates borders of chunks when coordinates are entered.

Minecraft Map & Pixel Art Simulator

This page is the conversion simulator for map and pixel art of minecraft.
It can use images that makes at image files, or the data that create in this app.
It also can add colors of new blocks that have map and base skin color.

3D Art Simulator for Buildings of Minecraft

This page has substantial of applications that can easily convert real images into 3D pixel art, which is useful when building at Minecraft game.
The flat image is then turned into frontal pixel art by adjusting the angle, and then into 3D art by using tools such as layer duplication.
And, the 3D display for confirmation is also available, so you can check the data in a way that is more similar to the real thing.
Finally, you can convert the resulting data into Excel data and download it as blueprint data with the placement of blocks per layer and the number of blocks required.
